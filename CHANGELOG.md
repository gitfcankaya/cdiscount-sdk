# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-01-15

### Added

- Initial release of CDiscount Octopia SDK for PHP
- OAuth2 authentication with client credentials flow
- Seller API support (seller info, addresses, indicators, subscriptions, carriers)
- Products API support (categories, brands, products, integration reports)
- Offers API support (packages, offer requests, offers management)
- Orders API support (orders, shipments, cancellations, commercial gestures)
- Order Invoices API support
- Discussions API support (discussions, messages, typologies)
- Fulfillment API support (inbound/outbound shipments, stocks, returns)
- Finance API support (invoices, operations, payments, reports)
- Comprehensive exception handling (ApiException, AuthenticationException, ValidationException)
- ApiResponse wrapper with pagination support
- Sample scripts for all API endpoints
- Complete documentation with usage examples
